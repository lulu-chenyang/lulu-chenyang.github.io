Kickoff

3 Types of Users

The products used in hotel is different from homes. The people who install them, use them, and mantain them are totally different. After talked with hotel staff and Stay Well team, I summarized users of Moodbox into the following 3 types:

Hotel Guest

45% of the visitors came to resort for pleasure. Their average stay is 3.4 nights to 4.4 days. More than 50% of the visitors travel long distance before they arrived.

Room service

Cleaning staffs need to rest the device every time after a guest checks out.

Commissioning team

Currently, the commissioning work is majorly done by Stay Well team. Potentially, in the future, when the business gets expand, a 3rd-party commissioning team will conduct the job.

Challenges

3 Different Needs

The Moodbox needs to be easily enough for hotel guests to control the light, set and enjoy the Dawn Simulation. It should not create extra workload for cleaning staff. The commissioning team should be able to commission all the devices in 5 min/room. It also needs to be ready for far more significant ancillary device controls through the cloud in the long term.

Dive deep

How Hotel Guests Control the Lights

The UX was developed based on existing Ketra's capability and user feedback. A motion active feature and Dawn Simulation feature were added. Along with a purposed system diagram designed by the team, the wireframe was quickly shipped to the factory to evaluate the feasibility.

Prototype

Multiple prototypes were developed during the development process for user testing. The "Set Alarm" button was replaced by a more accurate description "Set Dawn Simulation". The Stay Well blue was used for the focus button to improve brand integrity.

Users are able to set the time, set repeat, and select sound track. Once set, the user will experience natrual wake up 10 minutes before the set wake-up time.

Cleaning staff can reset the system by 2 clicks. Commissioning team can set up a light within 1 min.
